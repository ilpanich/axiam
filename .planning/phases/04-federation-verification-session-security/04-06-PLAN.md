    - `pub struct CleanupTask<C: Connection> { replay_repo: Arc<SurrealAssertionReplayRepository<C>>, state_repo: Arc<SurrealFederationLoginStateRepository<C>>, interval: Duration, shutdown: tokio::sync::watch::Receiver<bool> }`.
    - `pub fn new(...)` constructor.
    - `pub async fn run(mut self) -> Result<(), AxiamError>`:
      ```
      let mut ticker = tokio::time::interval(self.interval);
      ticker.set_missed_tick_behavior(tokio::time::MissedTickBehavior::Skip);
      loop {
          tokio::select! {
              _ = ticker.tick() => {
                  match self.replay_repo.cleanup_expired().await {
                      Ok(n) if n > 0 => tracing::debug!(deleted = n, "saml_assertion_replay cleanup"),
                      Ok(_) => {}
                      Err(e) => tracing::warn!(?e, "saml_assertion_replay cleanup failed"),
                  }
                  match self.state_repo.cleanup_expired().await {
                      Ok(n) if n > 0 => tracing::debug!(deleted = n, "federation_login_state cleanup"),
                      Ok(_) => {}
                      Err(e) => tracing::warn!(?e, "federation_login_state cleanup failed"),
                  }
              }
              changed = self.shutdown.changed() => {
                  if changed.is_ok() && *self.shutdown.borrow() {
                      tracing::info!("cleanup task received shutdown signal");
                      return Ok(());
                  }
              }
          }
      }
      ```

| T-04-37 | Tampering | Tests share state across invocations and produce flakes | accept | Each integration test builds its own in-memory SurrealDB via the existing test helpers; isolation per test |
| T-04-38 | Repudiation | A regression silently breaks REQ-5 / REQ-7 in a future phase | mitigate | Every AC has a named, traceable test — Phase 7 compliance phase reuses these as the conformance evidence |
| T-04-39 | DoS | wiremock mocks consume too many resources running 11 OIDC tests | accept | wiremock is per-test; cleanup is automatic when the mock guard drops; standard pattern in the ecosystem |
