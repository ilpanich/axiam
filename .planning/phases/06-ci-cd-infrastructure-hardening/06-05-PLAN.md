<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| pod ↔ pod (intra-namespace) | a compromised pod attempts lateral movement to others |
| pod → external network | a compromised pod attempts data exfiltration / C2 |
| workload spec → kubelet | a misconfigured pod requests excess privileges |
| manifest → cluster (secrets) | secret material could leak via inline manifest values |
