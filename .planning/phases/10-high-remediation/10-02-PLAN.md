Extract the four copy-pasted env-key loaders into one `load_key_from_env` helper, then use it to make PKI fail fast instead of silently encrypting CA private keys with an all-zero key. Closes REQ-14 (CQ-B43 + SEC-012).

CQ-B43 must land before SEC-012 because SEC-012 reuses the extracted helper.

Output: A single `load_key_from_env(name) -> Option<[u8;32]>` helper; all four blocks call it; `PkiConfig.encryption_key` becomes `Option<[u8;32]>`; CA/cert/pgp encrypt paths return a clear error when the key is absent.
