## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| anonymous → /auth/login | Timing side-channel can leak user existence |
| concurrent login attempts → user row | Race on failed-login counter |
| authenticated user → /api/v1 CRUD | State-changing requests need CSRF protection |
| authenticated user → self PUT /users/{id} | Privilege escalation via status/email tampering |
| authenticated user → /auth/logout | Session fixation / cross-session revocation |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-03-ENUM | Information Disclosure | service.rs login (SEC-026) | mitigate | Dummy Argon2 verify under spawn_blocking on not-found (ASVS V2) |
| T-11-03-TOCTOU | Tampering | failed-login counter (SEC-032) | mitigate | Atomic SurrealQL `+= 1` UPDATE (ASVS V2) |
| T-11-03-REUSE | Tampering | change_password (SEC-028) | mitigate | Reject reset-to-current with PasswordReusedCurrent (ASVS V2) |
| T-11-03-CSRF | Tampering | /api/v1 CRUD scope (SEC-046) | mitigate | CsrfMiddleware on api_scope; POST/PUT/DELETE without token → 403 (ASVS V4/V3) |
| T-11-03-AUTHZ | Elevation of Privilege | ROUTE_PERMISSION_MAP (SEC-047) | mitigate | Every mapped route has RequirePermission; register removed from PUBLIC_PATHS (ASVS V4) |
