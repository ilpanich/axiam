## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| in-process token storage → any log/debug sink | Token values held in memory must never cross into Debug/Display/tracing output |
| build.rs → filesystem (proto inputs) | Codegen reads proto files; a missing/tampered proto must fail loudly only when grpc is requested |

| T-16-02 | Information Disclosure | `AxiamError` variant messages | medium | mitigate | §2 mapping helpers take caller-controlled message only; documented prohibition on passing token values; no token field on any variant |
