## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SDK gRPC client → AXIAM gRPC service | Untrusted network; mTLS/TLS strict; auth + tenant metadata cross here |
| sync interceptor → async refresh guard | The sync interceptor must not block or await the async runtime |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|

| T-16-15 | Elevation of Privilege | wrong/absent x-tenant-id metadata | high | mitigate | UUID `x-tenant-id` (login-resolved) injected on every RPC; server cross-validates against claims (§5, authorization.rs:81-94) |
