| Boundary | Description |
|----------|-------------|
| Compromised server pod → arbitrary egress | Without tight egress rules a compromised pod can exfiltrate to any destination; default-deny + narrow allowlists limit blast radius |
| Operator config → running secret set | Missing secret keys degrade/fail features silently; the manifest must reference every key the app reads |
