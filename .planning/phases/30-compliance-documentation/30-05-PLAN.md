    Create scripts/check-doc-links.sh, an executable bash script (set -euo pipefail,
    matching the sibling scripts' style). It scans a fixed doc set — docs/**/*.md
    plus claude_dev/security-audit.md — extracts markdown inline links of the form
    label plus a parenthesized target, filters to RELATIVE targets only (skip
    http:/https:/mailto: and pure #anchor fragments), strips any trailing #anchor,

    correctly), and asserts each resolved path exists as a file. It prints each
    broken link with its source file and exits non-zero if any link is broken, exit
    0 if all resolve. Keep it stdlib-only (grep/sed/awk + test); no npm, no network.
    Make it executable (chmod +x). Self-test it during development against a known
    good doc and a temporary known-bad link to confirm it fails closed.
  </action>

    - `bash -n scripts/check-doc-links.sh` (syntax check) exits 0.
