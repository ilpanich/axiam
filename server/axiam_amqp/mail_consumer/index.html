Module mail_consumer

Expand description

AMQP consumer for asynchronous outbound mail delivery (D-14).

The consumer reads OutboundMailMessage from axiam.mail.outbound, resolves the effective [EmailConfig] via [EmailConfigRepository], renders the HTML-safe body via [render_email] (D-18), and sends via [EmailService].

On transient send failure the message is re-published with an incremented attempt_count for backoff retry. After MAX_RETRIES exhausted attempts the message is dead-lettered and a PII-minimal email.delivery_failed audit event is appended (D-16).
