GDPR Art. 15 (data export) and Art. 17 (erasure) REST endpoints.

Endpoints:

  • POST /api/v1/account/export — enqueue an async data-export job (D-12)
  • GET /api/v1/account/export/{token} — single-use download link (D-13)
  • POST /api/v1/account/delete — request account erasure / 30-day grace (D-07/D-08)
  • GET /api/v1/auth/account/delete/cancel?token=<opaque> — public cancel (D-09)

Constants§

GDPR_AUDIT_DLQ_FILE_ENV
Environment variable naming the append-only dead-letter file for the GDPR erasure audit event. Intended to point at a mounted volume so the file survives container restarts. Absent = the file sink is skipped for this event (the structured tracing event sink below still fires unconditionally on failure).
