Expand description

Injectable seam for the erasure audit DB-write (SECHRD-12).

SurrealAuditLogRepository<C> is the production implementation (forwards to AuditLogRepository::append); tests inject a failing double to drive the dead-letter path in write_erasure_audit_with_dlq without a live broken database. Defined here (not in axiam-core::repository) so it stays out of the generic repository trait surface owned by another plan.
