GET /.well-known/openid-configuration – OIDC Discovery document.

Returns the OpenID Provider metadata per OpenID Connect Discovery 1.0. The issuer URL is taken from AuthConfig::oauth2_issuer_url when set, falling back to AuthConfig::jwt_issuer otherwise.
