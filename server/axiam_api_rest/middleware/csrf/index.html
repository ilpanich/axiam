Module csrf 

Source
Expand description

CSRF double-submit cookie middleware and cookie builder helpers.

The CsrfMiddleware rejects state-changing requests that lack a valid X-CSRF-Token header matching the axiam_csrf cookie value. Comparison uses constant-time equality to prevent timing attacks (D-01).

Cookie helpers build the three auth cookies (axiam_access, axiam_refresh, axiam_csrf) with the security attributes specified in the UI-SPEC design decisions (D-05 through D-09).
