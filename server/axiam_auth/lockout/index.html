Shared failed-login / lockout accrual (D-06).

Both the REST login path (AuthService::record_failed_login) and the gRPC UserService::validate_credentials path call this single helper so that there is no unmetered credential-check path anywhere in the system (SEC-026b). This module is the single source of truth for the failed-attempt/lockout increment — do NOT duplicate this logic anywhere else.
