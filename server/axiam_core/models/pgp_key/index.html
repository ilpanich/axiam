Module pgp_key

Expand description

OpenPGP key domain models.

AXIAM manages two kinds of PGP keys:

  • AuditSigning: server-side signing keys with encrypted private keys stored for batch-signing audit log entries.
  • Export: zero-knowledge keys where only the public key is stored. The private key is returned once on generation.
