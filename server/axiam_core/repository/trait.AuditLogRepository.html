fn pseudonymize_actor( &self, tenant_id: Uuid, user_id: Uuid, pseudonym: &str, ) -> impl Future<Output = AxiamResult<u64>> + Send

Pseudonymize audit entries for a deleted user (D-03/D-04).

This is the ONLY non-INSERT write permitted on audit_log. It runs under the gdpr_pseudonymizer schema permission (v15).

Full D-03 scrub:

  • actor_id → nil UUID
  • metadata.actor_pseudonym → pseudonym string
  • ip_address → NULL
  • known PII metadata keys (email, username, name, etc.) → [redacted]
  • resource_id → nil UUID where it equals user_id

Returns the count of updated rows.
