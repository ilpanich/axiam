Expand description

Trait for publishing outbound mail messages to the async delivery queue.

Implemented by axiam-amqp’s MailOutboundPublisher. The thin trait lives in axiam-core so that axiam-audit (which must not depend on axiam-amqp due to the circular-dep constraint) can accept a generic publisher without coupling to the AMQP infrastructure layer.

Enqueue a single outbound mail message.

Implementations MUST be fire-and-forget at the call site: callers log a warning on error but do not propagate it to the client.
