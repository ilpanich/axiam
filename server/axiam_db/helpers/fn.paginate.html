Expand description

Assemble a PaginatedResult<T> from already-fetched items, the count_rows from a SELECT count() AS total ... GROUP ALL query, and the pagination window that was applied to the data query.

Replaces the let total = count_rows.first().map(|r| r.total).unwrap_or(0); Ok(PaginatedResult { items, total, offset, limit }) tail duplicated across every repo’s list method (CQ-B10).
