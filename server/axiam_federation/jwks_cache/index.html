Module jwks_cache 

Source
Expand description

JWKS cache with D-01/D-02/D-03 semantics.

  • D-01: 1-hour TTL — return cached keys if fetched_at + 1h > now.
  • D-02: Single forced refetch on unknown kid, rate-limited to 1 per 60 s.
  • D-03: 24-hour stale-while-revalidate — if the IdP is unreachable but the cache entry is no older than 24 h past TTL, serve stale keys with a WARN log rather than surfacing an error.
