Expand description

Federation client-secret encryption helpers (D-10..D-13).

Uses the split-output AES-256-GCM variant from axiam_auth::crypto (plan 04-01 Task 1) so each of client_secret_nonce and client_secret_ciphertext lands in its own DB column (D-11).

The bundled variant (aes256gcm_encrypt/aes256gcm_decrypt) is NOT used here — that is exclusively for TOTP secret storage.
