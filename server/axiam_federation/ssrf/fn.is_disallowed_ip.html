pub fn is_disallowed_ip(ip: IpAddr) -> bool
Expand description

Returns true for IP addresses that must never be contacted from a server-side outbound fetch to an admin/IdP-supplied URL.

Covers: RFC1918 private ranges, loopback (127/8 and ::1), link-local (169.254/16 and fe80::/10), broadcast (255.255.255.255), and unspecified (0.0.0.0 / ::). Lifted byte-identical from the two pre-existing duplicate copies (jwks_cache::is_private_jwks_ip, webhook::is_private_ip) — this is the D-01a dedup target.
